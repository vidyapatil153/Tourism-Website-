# 🌍 Explore Maharashtra - Tourism Website

A modern and responsive tourism website built using **HTML, CSS, and JavaScript** that showcases famous destinations across Maharashtra.

##  Features

-  Hero section with background travel video
-  Fully responsive navigation bar
-  Explore major cities of Maharashtra
-  Beaches and Forts pages
-  Reviews section
-  Newsletter subscription form
-  Social media links
-  Clean modern UI design

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts
